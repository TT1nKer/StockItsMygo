# 小资金动量交易指南 (2000 CAD)

## 📊 策略概述

针对2000加元小资金的动量交易策略，专注于短期强势股票。

### 核心优势
- ✅ **快速周转** - 持仓3-7天，资金不被长期占用
- ✅ **明确信号** - 清晰的入场和出场规则
- ✅ **集中火力** - 2-3只股票，便于管理
- ✅ **可控风险** - 单次亏损不超过35加元（总资金的1.75%）

---

## 💰 资金管理规则

### 基本设置
```
总资金: $2,000 CAD
单个仓位: $600-700 (30-35%总资金)
最多持仓: 2-3只股票
剩余现金: $600-800 (作为缓冲和机会资金)
```

### 风险控制
```
止损: -5% (每笔最多亏$35)
止盈: +12% (每笔盈利$84)
盈亏比: 1:2.4
最大回撤: 3笔连续止损 = -$105 (-5.25%总资金)
```

### 仓位计算示例
```python
# 股票价格 $15

可买股数 = 700 / 15 = 46股
实际投入 = 46 × 15 = $690

止损价 = 15 × 0.95 = $14.25
止盈价 = 15 × 1.12 = $16.80

最大亏损 = (15 - 14.25) × 46 = $34.50
预期盈利 = (16.80 - 15) × 46 = $82.80
```

## 🎯 选股标准

### 必须满足的条件

1. **价格区间**
   - 最低: $5 (避免垃圾股)
   - 最高: $100 (小资金买得起)
   - 推荐: $10-50 (最佳流动性)

2. **动量指标**
   - 20日涨幅 > 10% (强势)
   - 5日涨幅 > 0% (近期强势)
   - 最佳: 20日涨幅 > 20%

3. **成交量确认**
   - 近期成交量 > 1.5倍平均水平
   - 日均成交量 > 50万股
   - 最佳: 成交量放大2倍以上

4. **技术位置**
   - 价格在20日均线之上
   - 突破或接近近期高点
   - 无明显阻力位

### 评分系统 (0-100分)

```
优秀信号: 80-100分 - 强烈考虑
良好信号: 60-79分 - 可以考虑
一般信号: 40-59分 - 观察
弱信号: <40分 - 忽略
```

**评分构成:**
- 20日动量 (40分): >20% = 40分, >10% = 25分
- 成交量 (25分): >2倍 = 25分, >1.5倍 = 15分
- 突破确认 (20分): 突破 = 20分, 接近MA20 = 10分
- 近期强势 (15分): 5日涨幅 >3% = 15分
- 波动率调整 (-10分): 高波动 -10分

## 📋 交易执行步骤

### 开仓流程

1. **盘前准备**
   - 运行扫描脚本获取信号
   - 查看前3名高分股票
   - 确认技术图表形态

2. **选择标的**
   - 优先选择80分以上
   - 最多选择2只（分散风险）
   - 确保成交量充足

3. **下单执行**
   ```
   订单类型: 限价单
   买入价格: 当前价 ± 1%
   止损单: 同时设置止损 -5%
   ```

4. **持仓管理**
   - 每日盘后检查收盘价
   - 触发止损立即卖出
   - 盈利>8%可移动止损到成本价

### 平仓规则

#### 强制平仓（必须卖）
1. ❌ **触及止损** (-5%)
2. ❌ **成交量萎缩** (< 0.5倍平均)
3. ❌ **破位下跌** (跌破MA20且阴线)

#### 主动平仓（建议卖）
1. ✅ **达到止盈** (+12%)
2. ✅ **持有超过7天** (动量消退)
3. ✅ **出现更好机会** (且已盈利>5%)

#### 移动止损策略
```
盈利0-5%: 止损保持-5%
盈利5-8%: 移动止损到成本价
盈利8-12%: 移动止损到+5%
盈利>12%: 达到目标，考虑平仓
```

## ⚠️ 风险提示

### 必须遵守的铁律

1. **永远设置止损**
   - 没有止损 = 不开仓
   - 止损触发 = 立即执行

2. **不要追高**
   - 涨幅>50%且短期急涨 = 观望
   - 等待回调再进场

3. **控制仓位**
   - 单只股票不超过$700
   - 最多同时2-3只
   - 保留30%现金

4. **不要报复性交易**
   - 连续2次止损 = 休息1天
   - 重新分析市场

5. **记录交易日志**
   ```
   日期 | 股票 | 买入价 | 卖出价 | 盈亏 | 备注
   ```

### 常见错误

❌ **错误1**: 不设止损，期待反弹
✅ **正确**: 止损触发立即卖出，保护本金

❌ **错误2**: 一次all-in一只股票
✅ **正确**: 分散2-3只，单只不超过35%

❌ **错误3**: 盈利后贪心不卖
✅ **正确**: 达到12%目标立即止盈

❌ **错误4**: 频繁交易，一天多次进出
✅ **正确**: 持有3-7天，让动量发挥
